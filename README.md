# Awesome Sample Databases

Awesome Dockerized Pre-Filled Databases for Learning and Academic Projects

## Contributing

We welcome contributions! You can help improve this list in two ways:

- **Report an issue**: If you find a problem or want to suggest a new dockerized database, please [open an issue](https://github.com/ctrl-alt-d/AwesomeSampleDatabases/issues)
- **Submit a pull request**: If you want to add a dockerized database directly, please [create a pull request](https://github.com/ctrl-alt-d/AwesomeSampleDatabases/pulls)
